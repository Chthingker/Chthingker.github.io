# 课内

全过程课程： 计算机图形学（全英文） 编译原理 操作系统 通信原理

1-8 周： 汇编语言程序设计 毛泽东思想 云计算

11-18 周： 微型计算机接口技术 习近平新时代特色社会主义 就业指导

高数

英语

考研数学

# 运动

完成课内之后就去打球
调整状态后就去攻克自身的知识点

# 自身目标：

  • FMCW 雷达项目：心跳呼吸检测：
    这个重交流，我要清楚师兄师姐们在做什么，看论文资料全部整理好方便写作使用也清楚自己要做什么
    Matlab 官网的示例项目可以跟：信号处理 —— 人工智能
    跟进最新的论文： 技术 尤其是神经网络

  • AI 基础： 最后一学期：完全掌握：李沐的动手深度学习，PyTorch 以及机器学习基础完结：sklearn 以及 深度学习机器学习的笔记整理

  • 算法： 主要：搜索，树，等高阶的数据结构 以及相关算法题
    巩固方面：算法进阶课即可
    目标节点： 算法进阶课以及基础课刷完 加上：不定期的 CS 比赛

      书籍储存： 《数据结构编程》 《算法竞赛经典系列》《ACM的手册-题册等》 《挑战程序设计》

  打比赛太耗时间，不如积累

  • 软件： 继续 Go 语言为主： 深入，因为这是最近兴起的，被大家所接受的，而且我上手十分舒服的
    学习上：JavaWeb 全栈技术的学习

      Web应用就好了，特定操作系统的会有点小麻烦，而且这个通了那个相对成本高

  • 数模： 其实就是学习有用的数学知识，学习写作论文，能把有用的东西写出来：英文论文能力，等研究能力；抽象模型能力等；

      数模培训
  数模书籍等：
  最重要的是整理自己的论文以及所学知识以及应用：记录
